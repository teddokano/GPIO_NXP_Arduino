◆ begin()

void PCAL6416A::begin ( board env = NONE)

Device/board initialization

This method is needed to initialize Arduino-shield type evaluation boards from NXP. This method takes one argument of "PCAL6416A::ARDUINO_SHIELD" to set RESET and ADDRESS pins.

If the devoce is used as it self, this method doesn't need to be called.

Parameters
envThis argument can be given as "PCAL6416A::NONE" ot "PCAL6416A::ARDUINO_SHIELD"
